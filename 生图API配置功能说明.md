# 生图 API 配置模块 - 功能说明

## ✅ 已完成的集成

### 新增功能

1. **🎨 生图 API 配置按钮**
   - 位置：工具栏顶部
   - 功能：打开 API 配置模态窗口

2. **⚙️ API 配置模态窗口**
   - API 地址配置
   - API Key 配置
   - 模型选择
   - 默认尺寸设置
   - 图片质量设置

3. **🚀 一键生成图像**
   - 直接调用配置的 API
   - 自动使用分镜提示词
   - 在新窗口打开生成的图片

---

## 📋 功能详解

### 1. API 配置界面

**配置项：**

| 配置项 | 说明 | 示例 |
|--------|------|------|
| **API 地址** | OpenAI 兼容的 API 端点 | `https://api.openai.com/v1` |
| **API Key** | 你的 API 密钥 | `sk-...` |
| **模型名称** | 选择图像生成模型 | GPT Image-2 / DALL-E 3 / Imagen 3 |
| **默认尺寸** | 生成图片的尺寸 | 1024x1792（9:16 竖屏） |
| **图片质量** | 生成质量级别 | 高质量 / 中等 / 快速 |

**支持的模型：**
- ✅ GPT Image-2（推荐）
- ✅ DALL-E 3
- ✅ Gemini Imagen 3
- ✅ 自定义模型（如 Grok Aurora）

---

### 2. 使用流程

#### 步骤 1：配置 API

1. 点击工具栏的 **"🎨 生图API配置"** 按钮
2. 填写 API 地址和 Key
3. 选择模型和参数
4. 点击 **"测试连接"** 验证配置
5. 点击 **"保存配置"**

#### 步骤 2：生成分镜

1. 输入剧本
2. 点击 **"生成分镜与提示词"**
3. 查看生成的图像提示词

#### 步骤 3：生成图像

1. 点击 **"🎨 生成图像"** 按钮
2. 确认使用 API 生成
3. 等待生成完成
4. 图片会在新窗口中打开

## 🔧 技术实现

### 数据存储

**本地存储：**
```javascript
localStorage.setItem("imageApiConfig", JSON.stringify({
  url: "https://api.openai.com/v1",
  key: "sk-...",
  model: "gpt-image-2",
  size: "1024x1792",
  quality: "high"
}));
```

**安全性：**
- API Key 存储在本地浏览器
- 不会上传到服务器
- 仅在客户端使用

### API 调用

**请求格式：**
```javascript
POST /images/generations
Headers:
  Authorization: Bearer sk-...
  Content-Type: application/json

Body:
{
  "model": "gpt-image-2",
  "prompt": "分镜提示词...",
  "size": "1024x1792",
  "quality": "high",
  "n": 1
}
```

**响应处理：**
```javascript
{
  "data": [
    {
      "url": "https://...",
      "b64_json": "..."
    }
  ]
}
```

## 💡 使用建议

### API 选择

**GPT Image-2（推荐）：**
- ✅ 最新模型，质量最好
- ✅ 支持 4K 分辨率
- ✅ 多种比例支持
- 💰 价格：$0.04-0.20/张

**DALL-E 3：**
- ✅ 稳定可靠
- ✅ 兼容性好
- ⚠️ 分辨率有限
- 💰 价格：$0.04-0.08/张

**Gemini Imagen 3：**
- ✅ 多比例支持
- ✅ 质量优秀
- ⚠️ 需要 Google API
- 💰 价格：按使用量计费

### 尺寸选择

**分镜故事板：**
- 推荐：1536x2048（3:4）或 2048x1536（4:3）
- 原因：适合展示多个分镜

**单个分镜：**
- 推荐：1024x1792（9:16）或 1792x1024（16:9）
- 原因：适合视频制作

**高质量输出：**
- 推荐：3840x2160（4K）
- 原因：最高画质，适合专业制作

### 质量设置

**高质量（推荐）：**
- 生成时间：2-3 分钟
- 画质：最佳
- 适合：最终输出

**中等质量：**
- 生成时间：1-2 分钟
- 画质：良好
- 适合：预览测试

**快速生成：**
- 生成时间：30-60 秒
- 画质：可接受
- 适合：快速迭代

## 🔐 安全说明

### API Key 安全

**存储位置：**
- 本地浏览器 localStorage
- 不会上传到服务器
- 不会在网络中传输（除了调用 API）

**安全建议：**
1. ✅ 使用专用的 API Key
2. ✅ 定期轮换密钥
3. ✅ 设置使用限额
4. ✅ 监控 API 使用量
5. ❌ 不要分享 API Key
6. ❌ 不要在公共电脑上保存

### 隐私保护

**数据处理：**
- 提示词直接发送到配置的 API
- 不经过第三方服务器
- 生成的图片由 API 提供商存储

**注意事项：**
- 遵守 API 提供商的使用条款
- 不要生成违规内容
- 注意版权和隐私问题

## 🐛 故障排除

### 问题 1：配置保存失败

**原因：**
- 浏览器禁用了 localStorage
- 隐私模式下无法保存

**解决方法：**
- 使用正常浏览模式
- 检查浏览器设置
- 允许网站存储数据

### 问题 2：测试连接失败

**原因：**
- API 地址错误
- API Key 无效
- 网络连接问题
- CORS 跨域限制

**解决方法：**
- 检查 API 地址格式
- 验证 API Key 有效性
- 检查网络连接
- 使用支持 CORS 的 API

### 问题 3：生成图像失败

**原因：**
- API 额度不足
- 提示词违规
- 参数不支持
- 网络超时

**解决方法：**
- 检查 API 余额
- 修改提示词内容
- 调整生成参数
- 重试生成

### 问题 4：图片无法打开

**原因：**
- 浏览器阻止弹窗
- 图片 URL 过期
- 网络问题

**解决方法：**
- 允许浏览器弹窗
- 及时保存图片
- 检查网络连接
