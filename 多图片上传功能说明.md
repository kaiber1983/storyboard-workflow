## 🔧 使用说明

### 上传多张图片

**方法 1：一次性选择多张**
1. 点击"选择文件"按钮
2. 在文件选择器中按住 `Ctrl`（Windows）或 `Cmd`（Mac）
3. 点击选择多张图片
4. 点击"打开"

**方法 2：拖拽上传（如果浏览器支持）**
1. 选择多张图片文件
2. 拖拽到文件输入框区域
3. 释放鼠标

### 删除单张图片

1. 将鼠标悬停在要删除的图片上
2. 点击右上角的 `×` 按钮
3. 图片会立即从列表中移除

### 限制说明

- **最多上传：** 每个类型最多 9 张图片
- **超出处理：** 如果选择超过 9 张，只显示前 9 张，并显示警告提示
- **文件格式：** 支持所有图片格式（jpg, png, gif, webp 等）
- **文件大小：** 无前端限制，但建议单张不超过 10MB

## 📊 技术细节

### 文件管理

**使用 DataTransfer API 管理文件列表：**
```javascript
// 删除指定索引的文件
const dt = new DataTransfer();
Array.from(input.files).forEach((f, i) => {
  if (i !== index) dt.items.add(f);
});
input.files = dt.files;
```

### 预览生成

**使用 URL.createObjectURL 生成预览：**
```javascript
const url = URL.createObjectURL(file);
// 创建 <img> 标签显示预览
```

### 响应式布局

**自动适配不同数量的图片：**
- 1-3 张：显示 1 行
- 4-6 张：显示 2 行
- 7-9 张：显示 3 行
