### 步骤 3：在 Railway 部署 Image Studio

1. **访问 Railway**
   - 打开 https://railway.app
   - 使用 GitHub 登录

2. **创建新项目**
   - 点击 "New Project"
   - 选择 "Deploy from GitHub repo"
   - 选择 "image-studio" 仓库
   - 点击 "Deploy Now"

3. **生成域名**
   - 部署完成后，点击服务
   - 切换到 "Settings" 标签
   - 找到 "Networking" 部分
   - 点击 "Generate Domain"
   - 复制生成的域名（例如：`https://image-studio-production.up.railway.app`）

## 💡 使用提示

### 推荐的图像生成参数

**分镜故事板：**
- 尺寸：1536x2048（3:4）或 2048x1536（4:3）
- 质量：high
- 数量：1-2 张

**单个分镜：**
- 尺寸：1024x1792（9:16 竖屏）或 1792x1024（16:9 横屏）
- 质量：high
- 数量：1 张

**高质量输出：**
- 尺寸：3840x2160（4K）
- 质量：high
- 数量：1 张

## 🔧 故障排除

### 问题 1：按钮点击后没有反应
**解决方法：**
- 检查浏览器控制台是否有错误
- 确认提示词已生成
- 尝试刷新页面

### 问题 2：Image Studio 无法打开
**解决方法：**
- 检查 Railway 部署状态
- 确认域名配置正确
- 检查浏览器是否阻止了弹窗

### 问题 3：提示词复制失败
**解决方法：**
- 检查浏览器权限设置
- 手动复制提示词
- 使用 HTTPS 协议

### 问题 4：图像生成失败
**解决方法：**
- 检查 API 配置是否正确
- 确认 API 密钥有效
- 检查提示词格式
