💡 测试说明

1. 打开配置窗口：点击"打开生图API配置"按钮

2. 填写配置信息：

  • API 地址：例如 https://api.openai.com/v1
  • API Key：你的密钥（sk-...）
  • 选择模型和参数

3. 测试连接：点击"测试连接"验证配置

4. 保存配置：点击"保存配置"

5. 查看配置：点击"查看当前配置"查看已保存的信息

注意：配置保存在浏览器本地存储中，刷新页面后仍然保留。
